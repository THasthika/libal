#THB

A C library with useful data structures and algorithms

##Data Structures

- Linked List (Double)
- Stack
- Queue
- Hash Table
- Binary Search Tree

##Algorithms

###Sorting Algorithms

- Insertion Sort
- Merge Sort
- Heap Sort

###Installation

- run `./autogen.sh` in the repo directory
- run `./configure`
- run `make` to build the project
- run `make install` to install it to the system
