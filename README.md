### Dynamic Algorithms

- [x] Longest Common Subsequence

### Installation

1. run `mkdir build` in the repo directory
2. cd into the build dir `cd build`
3. run `cmake ../`
4. run `make` to build the project
5. run `make install` to install it to the system
